Firefox 目前可用的去广告方案（2021-10-02 更新）

此处的方案适用于 Firefox，所使用的扩展可能有 Chrome 版，Chrome 可参考，但不保证效果一致。

此处的方案所指的视频广告过滤，通常只针对优酷，爱奇艺，腾讯视频，其他视频站不一定有效。

此处的方案不一定是最好的方案。

211002 更新：修正 乘风视频规则 的地址。

210710 更新：Firefox 89 上下列方案仍然可用。

210124 更新：移除之前提到的 Nana Adblocker 的方案。

200303 更新方案：uBlock Origin + Header Editor

扩展 是否必需 备注
uBlock Origin 是 基础的广告过滤扩展，需要订阅扩展自带的 uBlock filters 和 乘风视频规则
Header Editor 否 参考卡饭的帖子设置优酷重定向：重定向规则1、重定向规则2

注：

  1. Header Editor 的重定向规则二选一即可。

    • 重定向规则1 会导致优酷的页面强制为网页全屏，不能显示评论和切换视频。
    • 重定向规则2 仅限 Firefox 57+ & 最新版的 Header Editor，优酷视频页面时会有一个短暂的重载，页面评论弹幕等功能均正常。

  2. 优酷可以不使用 Header Editor 的规则，uBlock Origin 的规则可以自动加速广告播放。

  3. 乘风视频规则 上文中提供的是 Github 的链接，如果无法访问的，也可以到卡饭的帖子找到新的链接。
