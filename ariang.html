AriaNg 网页参数切换 RPC 的参数

之前在Firefox 扩展 Aria2 下载器集成组件一文中提到，在 AriaNg 中，可以添加多个 RPC，通过点击页面的左上角进行切换，但是实际上切换操作上还是比较麻烦。

最近在 AriaNg 的 Github 上发现，可以通过配置网址参数的方式设定 RPC 的参数，所以，只需要保存两个配置好网址的书签，就可以通过点击不同的书签直接进入不同 RPC 的 AriaNg 界面。

  • AriaNg Default（端口为 6800，方式为 WebSocket）：

    http://ariang.mayswind.net/latest/#!/settings/rpc/set/ws/127.0.0.1/6800/jsonrpc

  • AriaNg Proxy（端口为 6801，方式为 WebSocket）：

    http://ariang.mayswind.net/latest/#!/settings/rpc/set/ws/127.0.0.1/6801/jsonrpc

下面仔细说明一下：

  • http://ariang.mayswind.net/latest/是 AriaNg Github 提供的 Demo，可以替换为自己的指定的网址。

  • settings/rpc/set/[RPC协议]/[RPC Host]/[RPC Port]/jsonrpc 是配置的参数，RPC 协议可选项包括 http、https、ws、wss，RPC Host 和 RPC Port 也可自行修改。

以上内容参考自 AriaNg 的文档，另外还有通过网址新建下载的参数。
