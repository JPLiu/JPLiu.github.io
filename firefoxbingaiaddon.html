在 Firefox 上启用 Bing AI 的对话机器人

请注意，本文只解决在 Firefox 浏览器上启用在默认情况无法打开 Bing AI 对话功能的问题，至于访问 Bing AI 的网络条件，不在本文范围。

Windows 端

  • 方案一：修改 Firefox 对 Bing 的 UserAgent，以下以 Header Editor 扩展为例 。

    • 安装 Header Editor 扩展。
    • 添加 Bing 的 Request 规则。
      • 名称：Bing Requset
      • 规则类型：修改请求头
      • 匹配类型：网址前缀
      • 匹配规则：https://www.bing.com
      • 排除规则：留空
      • 执行类型：常规
      • 头名称：user-agent
      • 头内容：填写 Edge 浏览器的 UserAgent，如 Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/107.0.0.0 Safari/537.36 Edg/107.0.1418.42 。

  • 方案二：使用 Bing AI for Firefox 扩展。

    • 安装 Bing AI for Firefox 扩展。
    • 安装后打开 Bing，登录即可使用 Bing AI 对话机器人功能。

Android 端

  • 方案一：使用 Bing AI for Firefox 扩展。
    • 安装 Bing AI for Firefox 扩展。
    • 安装后打开 Bing，登录即可使用 Bing AI 对话机器人功能。
