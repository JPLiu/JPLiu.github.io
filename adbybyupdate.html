一键更新广告屏蔽大师 ADbyby 的规则

2023.02.13 本文内容已过期，ADbyby 目前已无法正常使用。

广告屏蔽大师是近两年来使用的广告屏蔽软件，效果很不错，近期规则更新的服务器貌似被攻击，规则更新不正常，讨论群内的大佬给了个规则的更新地址，于是写了个脚本用来双击更新，放到广告屏蔽大师目录即可。

@echo off
:: 脚本使用aria2作为下载工具
set aria2=d:\1-Program\0-CommandLineTools\Aria2\aria2c.exe
:: 删除原有备份规则
del /s /q SSLLIST.ini.bak
del /s /q data\lazy.txt.bak
del /s /q data\video.txt.bak
:: 备份现有规则
move /y SSLLIST.ini SSLLIST.ini.bak
move /y data\lazy.txt data\lazy.txt.bak
move /y data\video.txt data\video.txt.bak
:: 指定下载地址
set sslistlink=https://raw.githubusercontent.com/adbyby/SSLLIST/master/SSLLIST.ini
set lazylink=https://raw.githubusercontent.com/adbyby/xwhyc-rules/master/lazy.txt
set videolink=https://raw.githubusercontent.com/adbyby/xwhyc-rules/master/video.txt

%aria2% --split=15 --max-connection-per-server=15 --dir=data %sslistlink%
%aria2% --split=15 --max-connection-per-server=15 --dir=data %lazylink%
%aria2% --split=15 --max-connection-per-server=15 --dir=data %videolink%

move /y data\ssllist.ini ssllist.ini
exit

攻击快点停吧，我懒得双击更新……
