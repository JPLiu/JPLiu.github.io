我的软件清单

  • 2021 年 10 月更新：

    • 调整 截图录屏 部分的 GIF 截图的内容。

  • 2021 年 7 月更新：

    • 增加 MPC-HC clsid2 版，一个很经典的本地视频播放器。
    • 补充 FFmpeg 的一个在 Github 上的 第三方 x86 编译版的链接。
    • 修改 Bulk Crap Uninstaller 的描述，现在有中文了。
    • 移除部分失效链接。

  • 2021 年二季度更新：

    • 增加 HiBit Uninstaller，一款单文件软件卸载工具。
    • 增加 Vidiot，一款简单的非线性视频编辑工具，无中文。
    • 增加 Wavosaur，一款音频编辑器，无中文，同时补充其导出 mp3 需要的 lame_enc.dll 的链接。
    • 分离音视频编辑栏目，更新 Audacity 的描述。
    • 移除文章最后长时间未更新的 googlehost。

  • 2021 年一季度更新：

    • 增加 Scrcpy，一款 通过 ADB 实现 Android 设备投屏到 Windows 上的软件。

下表是我个人目前在使用和曾经使用过的软件，大部分会提供链接，也可能会提供同类软件的替代品。软件名所进入的链接，全都是官方链接，第三方的链接会另外放出，由于我不会一直尝试曾经使用过的软件的新版本，描述可能与最新版本有所差异。

  • 文件管理
名称 功能
Total Commander 优秀的文件管理器，不过很多人无法习惯键盘操作
Double Commander 类 TC 的文件管理器，大体的功能都有，但是小细节上还有待提高，
支持部分 TC 的插件，汉化较完整
支持中文首字母搜索但是不完善（多音字只取一个读音，且不支持过滤）
Q-Dir 最多可以同时显示四个文件夹的小工具，
习惯用资源管理器而不习惯 TC 的用户可以用。
对高分屏支持不好。
FastCopy 拷贝工具，速度优势不大，但差分复制很好用，也有命令行参数可以调用，依然推荐。
WinMerge 开源，文件比较工具，
支持比较文本，图片，Word 文档，Excel 工作簿，PowerPoint 演示文稿。
Double Comander 的调用方法
  • 文件搜索
名称 功能
Everything NTFS 硬盘文件搜索利器，支持正则表达式。
可以当 Http 服务器使用，在手机上直接访问电脑上的文件。
Listary 文件搜索，定位利器，也可以当快速启动工具使用。
AnyTXT Searcher 全文检索工具，支持索引，搜索速度较快。
无便携版。
FileLocator 全文检索工具，分为免费的 Lite 版和收费的 Pro 版。
Lite 版不支持索引，搜索速度慢。
有便携版（Pro 版的便携版实际上通用，在初次打开时可选择只使用免费功能）
  • 剪贴板增强
名称 功能
CopyQ 开源，有中文，支持自定义命令，也支持简单截图功能，
支持 Vi 模式移动，便携
Ditto 开源，功能比 CLCL 要强，有UWP版，中文化不完整
Ditto 复制文件后无法在桌面这类特殊位置中粘贴
CLCL 开源，便携，免费的剪贴板工具，官方无中文。
  • 压缩解压
名称 功能
7-Zip 开源，无广告，也可以当双面板文件管理器使用，但是不支持查看 rar（非 rar5）格式的注释
PeaZip 开源，无广告，界面比 7-Zip 漂亮，有类似 BandiZip 的智能解压功能。
但是反应速度略慢，在命令行调用时，无法自行输入密码导致无法处理加密的压缩包。
  • 截图录屏
名称 功能
FastStone Capture 共享软件，官方只有英文版，同时支持截图与录屏。
ShareX 开源，截图软件，录屏和录制 GIF 功能通过 FFmpeg 实现（第一次使用会提示下载），
可以对截图进行在线 OCR，识别效果不错
Pickpic 从功能上来说也很不错，但它的滚动截图不支持自定义选择选区，
也无法设置自动保存文件并添加到剪贴版，只能二选一。
Gifcam GIF 截图工具，单文件，无中文。
ScreenToGif GIF 截图工具，开源，有中文。
  • 图片相关
名称 功能
IrfanView 图片查看软件，但不支持多标签，支持命令行转换，
也有很多插件（甚至有插件支持英文的 OCR）
XnView 图片查看软件，支持多标签，支持命令行转换，
有插件，对 Win 10 的支持比较好。
Paint.Net 免费图片编辑工具，便携化操作可参考 这里 的说明。
依赖 net.framework
GIMP；
PortableApps		 开源图片编辑工具，功能完整。
PhotoDemon 开源图片编辑工具，功能简单，
不需要 net.framework，但是中文翻译的界面有点乱。
LazPaint 开源图片编辑工具，功能简单，
不需要 net.framework，但是同时编辑多个文件不太方便。
  • 媒体相关
名称 功能
Foobar2000 本地音频播放工具，有第三方的汉化版
PotPlayer；
闻雷网盘		 本地视频播放软件，官方版开始有页游广告。
不过大部分人用的都应该是第三方的版本
VLC 本地视频播放软件，免安装版在目录下建一个portable文件夹可便携化配置。
MPC-HC clsid2 版 本地视频播放软件
MPC-BE 本地视频播放软件
K-Lite Codec Pack 解码器包，AviSynth 有些导入命令需要它支持，
WPS 用户安装了 Mage 版之后可以在 WPS 演示中直接播放视频。
MP3TAG；
PortableApps		 编辑音频文件的 tag，如按文件名自动填入 tag 或根据 tag 重命名文件，可批量处理
这个软件还有一个 TC 的插件。
  • 音视频编辑
名称 功能
AviSynth+ 视频编辑工具，通过写脚本实现编辑，再用 FFmpeg 生成结果文件，
功能比较少，但一般剪辑够用。
Vidiot 一款简单的非线性视频编辑工具，无中文。
Audacity；
PortableApps		 音频编辑工具，中文化不完整；
导入 ac3 等文件需要 FFmpeg 插件支持。
Wavosaur 音频编辑工具，无中文；
支持编辑的格式较少，比如不支持 m4a，可以使用 FFmpeg 等工具转换后处理，
导出 mp3 需要 lame_enc.dll 插件。
Subtitle Edit 字幕制作编辑工具，内置视频预览，翻译助理；
可调用 Google 在线翻译，可批量调整时间。
  • PDF相关
名称 功能
Sumatra PDF PDF 阅读器，功能单一，不支持加标注等功能；
支持 PDF；ePub；MOBI；XPS；DjVu；CHM；CBZ 和 CBR
可以当漫画查看器用
gImageReader 开源 OCR 工具，支持中文，英文等多种语言，
可见 gImageReader 简要介绍。
PDFCreator PDF 虚拟打印机，安装的时候要小心捆绑
PDFTK；
PDFTK Builder		 PDF 合并，分割小工具，PDFTK 的 GUI 免费版有功能限制，但是有命令行
PDFTK Builder 是该软件命令行的第三方 GUI，推荐使用
XPS to PDF 将 XPS 转换成 PDF，因为 Win 7 之后 Windows 自带了 XPS 的虚拟打印机，
用这个可以将 XPS 转换成 PDF，在一定程度上可以代替 PDFCreator 这种 PDF 虚拟打印机
PDF 相关在线服务 SmallPDF（多种处理）；ExtractPDF（提取 PDF 的图片，文本或字体）
  • 命令行相关
名称 功能
Aria2 通用下载工具
Youtube-DL 在线视频下载工具，支持清单
另可参看 WinPython 自动更新 You-Get 和 Youtube-dl
FFmpeg；
第三方 x86 编译版		 视频，音频格式转换工具
ImageMagick 图片转换工具，自带了 FFmpeg。
Pandoc 文档格式转换工具
安装包可用 7-Zip 解压，将两个名字最后是 EXE 的文件的后缀改成 EXE 即可调用，无需安装
Scrcpy 通过 ADB 实现 Android 设备投屏到 Windows。
WinPython 一个便携的 Python 环境，我用来更新的调用 You-Get 和 Youtube-DL。
iconv 文本编码转换工具
OpenCC 中文繁简转换工具
sed 文本编辑工具
awk 文本处理工具，我曾经用来提取 XX-Net 中速度较快的 ip

iconv、sed、awk 在 Git for Windows 中有集成，但是，Git for Windows 里面的 iconv 不指定源格式的话就没有办法转换。

  • 下载工具
名称 功能
Internet Download Manager 共享软件，相当不错的 HTTP/HTTPS/FTP 下载工具，支持 Firefox57+
uGet Download Manager 开源跨平台的下载软件，支持 HTTP/FTP/BT，支持 Curl 和 Aria2，可支持 Firefox57+ 的调用（需要配置）。
qBittorrent，
PortableApps，
第三方增强版		 BT下载工具，有 PortableApps 的便携版，
官方版可以通过在安装文件夹下新建 profile 文件夹实现便携化。
个人更建议使用增强版，可以拦截 迅雷等客户端。
WebTorrent BT 下载和边下边播工具，可以调用 VLC 播放器。
  • 清理、卸载工具
名称 功能
BleachBit 一款支持 winapp2.ini 的开源清理工具。
有中文（少部分未翻译），有便携版。
Bulk Crap Uninstaller 开源软件卸载工具，有便携版。
对于部分软件可以自动卸载（无需点击相关按钮）。
Geek Uninstaller 软件卸载工具，单文件。
HiBit Uninstaller 软件卸载工具，单文件，但是会把配置文件写到 C盘。
功能比 Geek Uninstaller 多，但是对高分屏支持不是太好。
  • 其它工具
名称 功能
AutoHotKey；
在线中文帮助文档		 热键，脚本编辑、工具，强烈推荐，特别是 RunAny。
chrlauncher 一个用来便携化 Chromium 的小工具，也支持将便携化的 Chromium 变为系统默认浏览器。
KeePass 密码管理工具，有第三方的安卓版 KeePassDroid 和 Keepass2Android 等。
NetsetMan 多套详细的网络设置快速切换，有命令行参数
LibreOffice；
ProtableApps		 开源办公套件，有便携版。
Visio Viewer 微软提供的免费的 Visio 文件（vsd，vdsx）的查看工具，安装后可以使用 IE 打开相关文件
Syncthing 开源多终端同步软件，支持便携化，我用来同步 Android 端和 Windows 端的文件。
Resource Hacker 资源修改工具，我用它来将 CCleaner 的增强规则整合到 CCleaner 内，官方无中文
小小输入法 外挂码表式输入法，无广告，内置五笔，郑码等码表
配置比较麻烦，需要启动程序后才能正常使用
All in One Runtimes 一个运行库和常用插件的组合包，不会锁定主页，但是没有中文
介绍可见 All in One Runtimes 简要介绍。
IT天空一键运行库 VC，Net，DirectX 运行库合集
有推广，使用时要小心
