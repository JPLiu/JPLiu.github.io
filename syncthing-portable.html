Syncthing 设置便携化

近期尝试了 Syncthing，实现在 Android 和 Windows 端同步或备份文件，以满足我在 Windows 上备份 Android 的微信接收的文件的要求，在中文的站点上我没有看到对于官方版的便携化方案，在这里补一下。

在 Windows 上 Syncthing 有三个选择：

  • 官方的 Syncthing

    • 命令行版本，启动后会打开一个命令行窗口，默认的配置文件在用户文件夹下，如果需要便携化的可以使用命令行参数。
    • 隐藏命令行窗口：-no-console
      • 例：syncthing.exe -no-console
    • 便携化：-home + 配置路径，其中配置路径可以是相对路径。
      • 例：syncthing.exe -home data：使用 Syncthing 所在路径下的 data 文件夹来保存和读取配置。
    • 以上参数可同时使用。

  • SyncTrayzor

    • Windows 的 Syncthing 包装器，默认就是便携的。

  • Syncthing-GTK

    • 跨平台的 Syncthing 包装器，我没有试用，此处不乱加说明。

在 Android 端上可以使用 Play 商店上的 Syncthing。
