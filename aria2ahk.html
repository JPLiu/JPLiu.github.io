Aria2AHK 简单介绍

由于后续脚本更新，本文内容可能与脚本最新情况存在差异。

2023.02.12 更新到 AutoHotkey v2，同时增加了 Aria2Torrent，用于关联 Torrent 后实现打开 Torrent 自动添加任务。

2022.08.20 简单更新，新增任务修改为使用 WebHttpRequest 实现。

2024.02.19 修改 GitHub 路径，说明以 GitHub 为准。

这是一个出现纯属偶然，后续维护完全没计划的 AutoHotkey 脚本。

切换到使用 Download with Aria2（通过 Aria2 下载） 扩展调用 Aria2 后，我一度回到使用 vbs 文件开机启动 Aria2 的方式，但由于默认下载路径在内存盘上，偶尔下载的文件（如 Windows 镜像），添加后由于文件过大无法正常下载，需要手工去启动存至另一路径的 Aria2，很不方便，于是昨晚趁着宿舍停水，拿 AutoHotkey 下写了个非常简单的 Aria2 托盘，这就是这里的 Aria2AHK。

  • GitHub 地址：Aria2AHK

大体功能

名称 功能 备注
打开 Web 前端 用浏览器打开本地的 Web 前端 路径在 ini 文件中设置
添加下载任务 添加下载任务
添加下载任务（代理） 同 添加下载任务，差异是会使用代理 代理在 ini 文件中设置
默认下载路径 修改 Aria2 默认的下载路径 默认路径在 ini 文件中设置，最多支持三个
选择后会自动重启 Aria2
更新 BT Tracker 更新 ini 文件中 BT Tracker 更新后会重启 Aria2
重启 Aria2 重启 Aria2
退出 退出 Aria2AHK 会同时退出 Aria2

主要的设置的说明我都写在了 Aria2AHK.ini 内

和之前几次 BT Tracker 直接更新到 Aria2 的 conf 文件不同，这次的 BT Tracker 参数保存在 Aria2AHK.ini 内，Aria2AHK 在运行 Aria2 时，会自动加载 Aria2AHK.ini 中保存的 Tracker。

Aria2AHK 的图标是我从 Download with Aria2（通过 Aria2 下载） 中翻出来的，如果有版权问题可以更换。
